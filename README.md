# Panadería Maldonado - Demo GitHub Pages

Versión estática compatible con GitHub Pages.

## Publicación

1. Subir todos los archivos de esta carpeta a la raíz del repositorio.
2. Verificar que `index.html` quede en la raíz.
3. En GitHub: Settings > Pages > Deploy from a branch > main > /root.

Esta versión simula una tienda tipo WordPress/WooCommerce con HTML, CSS y JavaScript. El proyecto ASP.NET/VB debe publicarse en hosting compatible con .NET/IIS, no en GitHub Pages.
