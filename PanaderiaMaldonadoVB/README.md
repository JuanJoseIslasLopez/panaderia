# Panadería Maldonado VB

Demo de tienda online estilo WordPress/WooCommerce desarrollada en ASP.NET Web Forms con Visual Basic.

## Páginas incluidas

- Inicio
- Tienda
- Carrito
- Checkout
- Confirmación de pedido

## Funcionalidades demo

- Catálogo de productos de panadería.
- Imágenes SVG locales para productos.
- Carrito en `localStorage`.
- Checkout con datos de facturación y envío.
- Métodos de envío estilo WooCommerce.
- Métodos de pago demo: efectivo, transferencia y Mercado Pago.
- Confirmación de pedido sin WhatsApp.

## Uso

Abrir el proyecto en Visual Studio y ejecutarlo como aplicación ASP.NET Web Forms.
